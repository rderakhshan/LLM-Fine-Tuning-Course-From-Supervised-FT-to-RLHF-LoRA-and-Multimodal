# LLM Fine-Tuning Course — From Supervised FT to RLHF, LoRA, and Multimodal

Interactive topic map visualizing the full LLM fine-tuning pipeline: unsupervised pre-training, supervised fine-tuning (SFT), preference alignment (RLHF/RLAIF/DPO), PEFT techniques (LoRA, QLoRA, DoRA, etc.), frameworks (Hugging Face, LLaMA Factory, Unsloth, Axolotl), and special topics (SLM, multimodal, embedding fine-tuning).

## Contents

- `llm-finetuning-topic-map.html` — Self-contained interactive HTML diagram with Mermaid mindmap and detailed CSS cards
- `README.md` — This file

## Viewing

Open `llm-finetuning-topic-map.html` in any modern browser. Dark/light mode adaptive.
